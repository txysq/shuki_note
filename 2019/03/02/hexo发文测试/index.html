leetcode – 到最近的人的最大距离

方法一：从左到右遍历找0和1的差值，只考虑右边的差。
从右到左遍历找0和1的差值，并与上一步的值进行比较，小的就是最近的人。
在最近的人数组中找到距离max

方法二：找到连续0的个数，考虑两个1中间的0 和首尾的0

第二种方法在内存和耗时上效果更好。
